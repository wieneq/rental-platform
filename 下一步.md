## 📝 接下來要做的事情：

### 第一步：安裝 Git（如果還沒有）

**檢查是否已安裝：**
1. 打開新的終端機（PowerShell）
2. 輸入：`git --version`
3. 如果顯示版本號，表示已安裝 ✅
4. 如果顯示錯誤，需要安裝 ⬇️

**安裝 Git：**
1. 訪問：https://git-scm.com/download/win
2. 下載並安裝（全部使用預設選項）
3. **重要：安裝完成後關閉並重新打開 VS Code**

---

### 第二步：使用 VS Code 上傳到 GitHub

這是最簡單的方法，不需要輸入任何命令！

#### 2.1 初始化 Git
1. 點擊 VS Code 左側的「源代碼管理」圖標
   - 圖標像三個圓點連成的樹枝
   - 快捷鍵：`Ctrl + Shift + G`
2. 點擊藍色按鈕「初始化存儲庫」

#### 2.2 提交代碼
1. 會看到很多文件列在「更改」中
2. 點擊「更改」旁邊的「+」號（暫存所有更改）
3. 在上方輸入框輸入提交訊息：
   ```
   初始提交 - 共享器材租賃管理平台
   ```
4. 點擊「✓ 提交」按鈕

#### 2.3 發布到 GitHub
1. 點擊「發布分支」按鈕
2. 選擇存儲庫類型：
   - **Public**：任何人都可以看到（推薦用於展示）
   - **Private**：只有您可以看到
3. VS Code 會要求登入 GitHub
   - 如果沒有帳號，先去 https://github.com 註冊
   - 如果有帳號，按照提示登入授權
4. 等待上傳完成 ✅

**確認成功：**
- 訪問 https://github.com/您的用戶名
- 應該可以看到新的存儲庫

---

### 第三步：部署到 Vercel

#### 3.1 創建 Vercel 帳號
1. 訪問：https://vercel.com
2. 點擊「Sign Up」
3. **選擇「Continue with GitHub」**
4. 授權 Vercel 訪問您的 GitHub

#### 3.2 導入專案
1. 點擊右上角「Add New...」
2. 選擇「Project」
3. 找到您剛才上傳的存儲庫（名稱：`20260322共享器材租賃管理平台`）
4. 點擊「Import」

#### 3.3 配置專案

**基本設定：**保持預設即可

**環境變數（重要！）：**
1. 展開「Environment Variables」
2. 添加一個環境變數：
   - Name: `SESSION_SECRET`
   - Value: `rental-platform-secret-key-2026`
3. 點擊「Add」

#### 3.4 開始部署
1. 點擊底部大大的「Deploy」按鈕
2. 等待 1-2 分鐘
3. 看到 🎉 慶祝動畫就成功了！
4. 點擊「Visit」查看您的網站

每次修改代碼後：

## ❓ 常見問題

### Q1: Git 安裝後還是顯示找不到？
**A:** 重啟 VS Code，或重啟電腦

### Q2: GitHub 登入失敗？
**A:** 確認已註冊 GitHub 帳號，使用瀏覽器登入測試

### Q3: Vercel 部署失敗？
**A:** 
1. 檢查環境變數是否正確設置
2. 查看 Vercel 的部署日誌找錯誤訊息
3. 確認 GitHub 存儲庫有 vercel.json 文件

### Q4: 網站訪問後顯示 404？
**A:** 
1. 等待幾分鐘讓部署完成
2. 訪問 `https://您的網址.vercel.app/index.html`
3. 重新部署專案

### Q5: 登入後資料消失？
**A:** 這是正常的！目前使用記憶體儲存，每次部署後資料會重置。這是展示版的特性。
